## Responsive Behavior

### Desktop (1920px)
- Gallery: 4 columns
- All filters in one row
- Side-by-side stats boxes
- Full table visible

### Tablet (768px)
- Gallery: 2-3 columns
- Filters may wrap to 2 rows
- Stats boxes stack on smaller tablets
- Table scrolls horizontally

### Mobile (375px)
- Gallery: 1 column
- Filters stack vertically
- Stats boxes stack
- Table scrolls horizontally
- Simplified preview dialog
- Touch-friendly buttons (larger tap targets)
